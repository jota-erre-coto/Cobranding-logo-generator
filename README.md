# Cobranding Logo Generator

Una herramienta web para generar logos de co-branding componiendo el logo del partner con el tagline "by civitatis".

## Características

- 🎨 **Composición flexible**: Modo horizontal y vertical (mixta en móvil)
- 📱 **Responsive**: Previsualización para desktop y móvil
- ⚙️ **Controles avanzados**: Ajusta espaciado, escala y más
- 📦 **Exportación múltiple**: SVG, PNG (@1x y @2x) en ZIP
- 🎯 **Smart Presets**: Compact, Balanced, Airy con recomendaciones automáticas
- 🏷️ **Tagline fijo**: El tagline mantiene tamaño consistente sin importar la escala del partner

## Uso

1. Abre `index.html` en tu navegador
2. Sube un logo SVG del partner
3. Ajusta la composición y espaciado según necesites
4. Previsualiza en desktop y móvil
5. Exporta como SVG o descarga el ZIP con assets

## Requisitos

- Navegador moderno (Chrome, Firefox, Safari, Edge)
- Logo del partner en formato SVG

## Tecnologías

- HTML5, CSS3, JavaScript vanilla
- Montserrat font (Google Fonts)
- JSZip para empaquetado de archivos
